# HappyGingerFE

## To get started

- git clone https://github.com/thehappyginger/HappyGingerFE.git
- cd HappyGingerFE
- npm install
- yarn install

## To start Dev Server:

- yarn run build
- yarn run start

## To start Production Server

- yarn run build -- --release
- yarn run start -- --release
